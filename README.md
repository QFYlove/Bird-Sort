# Bird-Sort
# 🐦 Bird Sort - 暂存格功能原型 (产品助理测试题)

## 📝 项目简介
本项目为《Bird Sort》游戏“暂存格 (Holding Nests)”功能的产品需求文档 (PRD) 附属验证原型。
为了验证 PRD 中设计的核心玩法逻辑与商业化流程的技术可行性，本人利用 AI 辅助编程 (Vibe Coding) 独立开发了此 Python 极简交互原型。

通过该原型的实际运行，验证了业务逻辑在底层数据流转上的闭环，排除了潜在的逻辑死局与数据冲突。

## ✨ 核心已验证逻辑
* **同色消除规则：** 树枝满 4 只同色小鸟自动消除。
* **暂存区调度：** 严格遵循“单只移入”与“同色/空枝移出”的双向校验规则。
* **商业化状态流转：** 模拟 AD 锁状态，包含解锁前后的逻辑放行与拦截。
* **作弊模式验证：** 提供无视规则的单步强制移动功能，用于边界情况测试。
